# E.M.A — Expert Makeup Academy

Next.js site for Expert Makeup Academy (white / light theme).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```

## Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
